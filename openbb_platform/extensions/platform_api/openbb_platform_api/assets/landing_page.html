Connect to the OpenBB Workspace

Connect your OpenBB Platform backend with OpenBB Workspace to access all your data in an enterprise UI built for AI workflows.

Steps to Integrate with OpenBB Workspace

  1. Sign in to OpenBB Workspace
  2. Navigate to the "Apps" tab
  3. Click on "Connect backend"
  4. Fill in the form with:
    • Name: OpenBB Platform
    • URL: http://127.0.0.1:6900
  5. Click on "Test" - You should see "Test successful" with the number of apps found
  6. Click on "Add" to complete the integration

For more detailed information about integrating OpenBB Platform with OpenBB Workspace, including advanced configuration options and troubleshooting, visit our comprehensive integration guide .

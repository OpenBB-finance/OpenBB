## Added

Commands:

### Stocks:
- /markets - Provides a nice overview of the market with adv / decliners relative to change and 50d/200d MA
- /scr pm_all - Displays all available pre-market screeners
- /scr ah_all - Displays all available after-hours screeners

### Crypto:
- /crypto tvl - Get protocols historical TVL.
- /crypto defi table - Get top DeFi protocols
- /crypto defi chart - Chart of top 20 defi protocols by TVL
- /crypto top - Get top cryptos and info
- /crypto bio - Get fundamental information about a crypto.
- /crypto fees - Fees on a certain date of top crypto protocols
- /crypto compfees - Get protocol fees over time.
- /crypto ir - Returns a table of issuance rate of top crypto protocols.

### NFT:
- /nft top - Get Top NFT Collections from Ethereum Chain
- /nft collection - Get NFT collection data from Ethereum network for the specified slug.
- /nft chart - Returns NFT chart for floor price and sales on the specified slug.

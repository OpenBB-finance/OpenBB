# rs

Display a repo summary [Source: https://api.github.com]

### Usage

```python
rs -r REPO
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| repo | Repository to search for repo summary. Format: org/repo, e.g., openbb-finance/openbbterminal | None | False | None |

# toplobbying

Top lobbying. [Source: www.quiverquant.com]

### Usage

```python
toplobbying [-l LIMIT] [--raw]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display | 10 | True | None |
| raw | Print raw data. | False | True | None |

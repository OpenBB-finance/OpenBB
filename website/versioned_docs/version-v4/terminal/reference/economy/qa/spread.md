# spread

Shows rolling spread measurement

### Usage

```python
spread [-w N_WINDOW]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| n_window | Window length | 14 | True | None |

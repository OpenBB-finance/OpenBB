# stats

Display coin stats

### Usage

```python
stats [--vs {USD,USDC,USDT,EUR,USD,GBP}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| vs | Quote currency (what to view coin vs) | USDT | True | USD, USDC, USDT, EUR, USD, GBP |

# overview

Show overview of funds from selected country.

### Usage

```python
overview [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of search results to show | 10 | True | None |

# set

Select similar companies to compare with.

### Usage

```python
set [-s L_SIMILAR]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| l_similar | similar companies to compare with. |  | True | None |

# hist

Histogram with density and rug

### Usage

```python
hist [-b N_BINS]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| n_bins |  | 15 | True | None |

# psi

Shows price vs short interest volume. [Source: Quandl/Stockgrid]

### Usage

```python
psi [--nyse]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| b_nyse | Data from NYSE flag. Otherwise comes from NASDAQ. Only works for Quandl. | False | True | None |

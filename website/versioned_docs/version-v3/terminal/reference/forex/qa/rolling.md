# rolling

Rolling mean and std deviation

### Usage

```python
rolling [-w N_WINDOW]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| n_window | Window length | 14 | True | range(5, 100) |

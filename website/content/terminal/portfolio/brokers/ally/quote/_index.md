```
usage: quote [-t TICKER] [-h]
```
Get stock quote
```
optional arguments:
  -t TICKER, --ticker TICKER
                        Ticker to get quote for. Can be in form of 'tick1,tick2...' (default: None)
  -h, --help            show this help message (default: False)
```

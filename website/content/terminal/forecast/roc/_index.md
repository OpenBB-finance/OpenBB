```
usage: roc [-d {}] [-c TARGET_COLUMN] [--period PERIOD] [-h]
```

Add rate of change to dataset based on specific column.

```
optional arguments:
  -d {}, --target-dataset {}
                        The name of the dataset you want to select (default: None)
  -c TARGET_COLUMN, --target-column TARGET_COLUMN
                        The name of the specific column you want to use (default: close)
  --period PERIOD       The period to use (default: 10)
  -h, --help            show this help message (default: False)

For more information and examples, use 'about roc' to access the related guide.
```

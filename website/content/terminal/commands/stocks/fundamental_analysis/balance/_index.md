```
usage: balance [-l LIMIT] [-q] [-h] [--export {csv,json,xlsx}]
```

Prints a complete balance sheet statement over time. This can be either quarterly or annually. [Source: Alpha Vantage]

```
optional arguments:
  -l LIMIT, --limit LIMIT
                        Number of latest years/quarters. (default: 1)
  -q, --quarter         Quarter fundamental data flag. (default: False)
  -h, --help            show this help message (default: False)
  --export {csv,json,xlsx}
                        Export raw data into csv, json, xlsx (default: )
```

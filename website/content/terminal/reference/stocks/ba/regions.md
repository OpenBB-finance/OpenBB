# regions

Plot bars of regions based on stock's interest. [Source: Google]

### Usage

```python
regions [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | limit of regions to plot that show highest interest. | 10 | True | None |

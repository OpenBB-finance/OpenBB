# dupont

The extended dupont deconstructs return on equity to allow investors to understand it better

### Usage

```python
dupont [--raw]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| raw | Print raw data. | False | True | None |

# divs

Historical dividends for a company

### Usage

```python
divs [-l LIMIT] [-p]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of previous dividends to show | 16 | True | None |
| plot | Plots changes in dividend over time | False | True | None |

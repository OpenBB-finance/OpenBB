# alswe

Show allocation of a swedish fund.

### Usage

```python
alswe [--focus {all,country,sector,holding}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| focus | The focus of the funds exposure/allocation | all | True | all, country, sector, holding |

# gdapps

Display top dApps (in terms of TVL) grouped by chain. [Source: https://docs.llama.fi/api]

### Usage

```python
usage: gdapps [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of top dApps to display | 40 | True | None |

# sreturn

Displays terra blockchain staking returns history. [Source: https://fcd.terra.dev/swagger]

### Usage

```python
sreturn [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of days to show | 90 | True | None |

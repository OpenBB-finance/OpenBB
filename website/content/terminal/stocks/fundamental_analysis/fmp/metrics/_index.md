```
usage: metrics [-l LIMIT] [-q] [-h] [--export {csv,json,xlsx}]
```

Prints a list of the key metrics of a company over time. This can be either quarterly or annually. [Source: Financial Modeling Prep]

```
optional arguments:
  -l LIMIT, --limit LIMIT
                        Limit of latest years/quarters. (default: 1)
  -q, --quarter         Quarter fundamental data flag. (default: False)
  -h, --help            show this help message (default: False)
  --export {csv,json,xlsx}
                        Export raw data into csv, json, xlsx (default: )
```

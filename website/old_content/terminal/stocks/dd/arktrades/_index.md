```
usage: arktrades [-l LIMIT] [-s] [-h] [--export {csv,json,xlsx}]
```

Get trades for the loaded ticker across all ARK funds.

```
optional arguments:
  -l LIMIT, --limit LIMIT
                        Limit of rows to show
  -s, --show_ticker     Flag to show ticker in table
  -h, --help            show this help message
  --export {csv,json,xlsx}
                        Export raw data into csv, json, xlsx
```
